# layout

用途：与 layout 相关的前端实现。

## 说明
- 归属路径：electron\src\components\layout
- 修改本目录代码后请同步更新本 README

## 策略卡片展开动画（2026-02 更新）
- `ModuleGrid` 新增策略模块放大交互容器：
  - 点击策略卡片“查看所有策略”后，使用轻量模态缩放动画展示全屏浮层（`92vw x 86vh` 上限）。
  - 显示半透明遮罩并支持点击遮罩关闭。
  - 支持 `Esc` 关闭。
  - 展开期间锁定 `document.body` 滚动，关闭后恢复。
- 动画实现基于 `framer-motion` 的 `AnimatePresence`（淡入遮罩 + 缩放面板），避免共享布局测量卡顿。
- 视觉实现采用轻毛玻璃遮罩：
  - 背景层：低透明度深色遮罩 + `backdrop-blur(6px)` + 轻度饱和增强。
  - 弹窗层：半透明白底、细边框与柔和投影，突出前景不压灰背景。
- 风格补充：
  - 降低遮罩深度，避免弹窗内容显得发灰。
  - 入场/退场曲线改为更柔和的缩放淡入。

## 通知卡片展开动画（2026-02 更新）
- `ModuleGrid` 同步支持通知模块弹窗展开（与策略模块同款交互）：
  - 点击“查看所有通知”后弹出中间层面板。
  - 支持遮罩点击关闭与 `Esc` 关闭。
  - 展开期间共用页面滚动锁定逻辑。
- 动画实现同样基于 `AnimatePresence` 的淡入遮罩 + 缩放面板。

## 通知中心页接入（2026-03-09）
- `DashboardLayout` 新增 `notifications` Tab 渲染分支，展示 `modules/NotificationCenter`。
- 与仪表盘卡片并存：
  - 仪表盘卡片用于快速预览与快捷操作；
  - 通知中心页用于完整筛选、已读处理与链接跳转。

## 顶部回调（2026-04-11）
- `HeaderBar` 已回调为更简约的三栏顶部：左侧连接状态，中部品牌标题，右侧时间信息，避免首页头部层级过重。
- `ModuleGrid` 的内容区恢复为常规 `p-6`，并同步恢复头部占位高度，减少顶部视觉体积。
- 在简约结构上补做了轻量精修：状态胶囊和时钟改为更轻的玻璃质感，标题字距与副标 tracking 重新收敛，整体更细致但仍保持克制。
