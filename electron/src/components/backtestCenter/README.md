# backtestCenter

用途：回测中心相关功能。

## 说明
- 归属路径：electron\src\components\backtestCenter
- 修改本目录代码后请同步更新本 README

## 近期更新
- 回测中心导航与模块页标题统一（2026-04-10）：
  - 左侧导航中的 `快速回测 / 专家模式 / 参数优化 / 策略管理` 已更新为标准标题；
  - 面包屑导航同步切换，和具体模块页头保持一致；
  - `策略管理` 与 `高级分析` 的模块页标题也同步更新，保持口径一致。
- 回测中心分析页配色统一：
  - `BacktestHistoryModule`、`EnhancedAdvancedAnalysisModule` 的概览卡改为红涨绿跌、风险项暖色、计数项中性灰；
  - 快速回测结果页与分析页保持同一套 A 股风格口径，减少跨页面颜色认知成本。
- 参数优化历史入口增强（2026-03-10）：
  - 参数优化模块增加右侧“优化历史”侧栏，支持查看最近记录、恢复运行中任务、查看失败原因与回填历史最优参数。
  - 参数优化页面重新进入后会优先恢复最近的运行中 Grid Search，不要求用户留在当前页等待完成。
- 参数优化模块降级为“受限可用（Beta）”：
  - 页面入口改为稳定模式，仅开放网格搜索；
  - 启动优化前检查引擎队列可用性，不可用时直接拦截；
  - 优化结果支持一键回填 `qlib_strategy_params(topk/n_drop)` 到回测中心配置，方便快速回测复用。
- 高级分析页面重构为“后端可用能力优先”：
  - 移除未落地的“绩效分析”入口，仅保留 `基础风险 / 交易统计 / 基准对比`。
  - 临时移除“持仓分析”入口（后端持仓解析存在稳定性问题），避免影响高级分析主流程。
  - 移除多个“待补充/占位”卡片，页面仅展示后端真实可返回的数据模块。
  - 选择回测后首栏固定 5 个概览指标：`总收益率 / 夏普比率 / 最大回撤 / 投资胜率 / 交易次数`，并与下方 3 模块栅格保持边距对齐。
- 回测详情标题改为“策略名称 | 回测期间”，不再展示指数名；回撤曲线悬浮提示统一为百分比并保留 2 位小数。
- 回测详情标题与配色修正：
  - 标题标的字段不再回退使用 `universe`（如 `csi300`），优先展示 `symbol/config.symbol/benchmark_symbol`，并将 `csi300` 规范显示为 `沪深300 (SH000300)`。
  - 按中国股市风格调整颜色：上涨为红色、下跌为绿色；权益曲线改为红色，回撤曲线改为绿色。
- 回测详情弹窗信息修复：
  - 详情副标题从 `symbol/start_date/end_date` 增加 `config` 回退读取，避免显示 `- | - ~ -`。
  - 权益曲线金额轴增加格式化（万/亿单位，保留 2 位小数），tooltip 金额统一保留 2 位小数。
- 策略对比模块 `user_id` 解析逻辑与回测历史统一：优先使用登录态用户（`authService.getStoredUser()`），避免因 `default_user` 查询口径导致对比列表缺失最新记录
- 回测历史模块补齐详情弹窗入口（基于 BacktestHistory 的 onViewDetail 回调）
- 修复回测历史为空问题：历史查询 `user_id` 改为优先取当前登录用户（`authService.getStoredUser()`），避免固定 `default_user` 导致查询不到真实回测记录。
