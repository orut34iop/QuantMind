# community

用途：社区相关功能。

## 说明
- 归属路径：electron\src\components\community
- 社区头部用户名显示优先读取 Redux `profile.username`，若无再回退认证用户快照，确保个人中心改名后社区页同步更新。
- 社区详情页支持分享深链参数 `communityPostId`：
  - 社区列表页初始化会读取 URL 参数并直接打开帖子详情；
  - 详情页返回列表或关闭详情时会自动清理该参数。
- 分享按钮已统一输出社区深链（复制链接/微博分享都带 `communityPostId`），避免分享后落到错误页面。
- “关注作者”按钮已接入后端真实关系接口，状态与粉丝数由服务端返回为准（支持关注/取消关注与详情页状态回显）。
- 社区主界面“活跃达人”支持头像加载：
  - 优先展示后端 `hotUsers.avatar`；
  - 相对路径头像会自动拼接 API 网关域名；
  - 头像加载失败自动回退为用户名首字母占位，不影响列表渲染。
- 修改本目录代码后请同步更新本 README
