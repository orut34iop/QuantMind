# utils

用途：工具函数与通用辅助。

## 说明
- 归属路径：electron\src\utils
- 修改本目录代码后请同步更新本 README
- `excelExport.ts` 已新增交易记录 Excel 导出能力，可与交易页的 CSV/Excel 双格式导出配合使用。
- `chartOptions.ts` 不再自行按自然日推导横轴窗口，而是直接消费 Hook 按交易日历补齐后的序列：
  - 每日收益图使用最近 30 个交易日；
  - 交易次数图使用最近 7 个交易日，并维持整数步进纵轴。
- `format.ts` 新增后端时间解析辅助函数：无时区时间会按 UTC 解释，再统一按上海时区展示，避免交易记录/更新时间出现 8 小时偏移。
