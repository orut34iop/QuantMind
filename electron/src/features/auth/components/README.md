# components

用途：可复用的 UI 组件。

## 说明
- 归属路径：electron\src\features\auth\components
- 修改本目录代码后请同步更新本 README

## 变更记录

- 登录页减少卡片与按钮动画，固定登录框位置，仅保留背景渐变动画以降低资源消耗。
- 注册页字段调整：首行由邮箱改为用户名输入（映射 `full_name`），原“真实姓名（可选）”输入改为邮箱输入（映射 `email`）。
- 注册页校验调整：除邮箱外其余字段均为必填，用户名新增必填与空白字符拦截校验。
- 注册页交互调整：获取短信验证码按钮增加用户名前置校验，用户名为空时按钮禁用。
- 注册页验证码体验调整：新增重发倒计时提示文案，并在冷却期内重复触发时给出限流提示。
- 注册页错误处理增强：新增后端短信错误码/状态码映射（429 限流、503 配置异常、日限额等）并输出友好文案。
- 登录页字段文案调整：密码登录账号输入由“用户名或邮箱”改为“用户名或手机号”。
- 登录页加载后新增 AI-IDE 预加载：会在空闲时异步预取 `/ai-ide` 页面资源并预热 Electron 侧 AI-IDE 运行时状态，开发/生产模式均生效。
- 登录页视觉修复：统一 `login-card` 外层/内层圆角裁剪与背景裁切，修复 Windows/Electron 下四角偶发白边问题。
- 认证页视觉一致性增强：注册/忘记密码/重置密码/MFA 验证页统一接入 `auth-rounded-card` 圆角裁剪样式，降低同类白边问题复发概率。
