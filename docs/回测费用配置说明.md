# 回测费用配置说明

## 更新日期
2026-02-21

## 重要说明

**费用计算方式**: 本系统采用**费率公式**，而非固定金额；当前后端按分项真实费用计算（佣金/过户费/印花税/最低费用/冲击成本）。

```
买入费用 = max(成交金额 × commission, min_commission) + 过户费 + 冲击成本
卖出费用 = max(成交金额 × commission, min_commission) + 过户费 + 印花税 + 冲击成本
```

**费率是按比例收取的**，不是固定金额！

## 费用优化说明

基于 `docs/费用.md` 的真实A股交易费用结构，已将回测系统的交易费率调整为更符合实际的值。

## 真实A股费用结构

### 费用组成（按成交金额比例收取）

| 费用项目 | 费率 | 收取方式 | 说明 |
|---------|------|----------|------|
| **券商佣金** | 0.025% (万2.5) | 买卖双向 | 最低5元/笔 |
| **过户费** | 0.001% (万0.1) | 买卖双向 | 已并入全市场 |
| **印花税** | 0.05% (万5) | 仅卖出 | 2023年8月28日起减半 |
| **冲击成本** | 默认系数0.0005 | 买卖双向 | 与成交参与率相关（非线性） |

### 当前后端默认参数（已核对代码）

```
commission = 0.00025
min_commission = 5.0
stamp_duty = 0.0005
transfer_fee = 0.00001
min_transfer_fee = 0.01
impact_cost_coefficient = 0.0005
```

> 说明：`open_cost/close_cost` 仍可作为兼容别名传入（映射到 `buy_cost/sell_cost`），但推荐使用上面的分项参数口径。

### 前端默认费率（本次调整）

```
默认佣金费率 = 0.00025（万2.5）
```

## 费用计算示例

### 示例1: 交易10万元

**买入费用**:
- 佣金: 100,000 × 0.025% = 25元
- 过户费: 100,000 × 0.001% = 1元
- **合计**: 26元

**卖出费用**:
- 佣金: 100,000 × 0.025% = 25元
- 过户费: 100,000 × 0.001% = 1元
- 印花税: 100,000 × 0.05% = 50元
- **合计**: 76元

**总交易成本**: 26 + 76 = **102元** (占本金0.102%)

### 示例2: 交易1万元（触发最低佣金）

**买入费用**:
- 佣金: max(10,000 × 0.025%, 5元) = 5元（最低5元）
- 过户费: 10,000 × 0.001% = 0.1元
- **合计**: 5.1元

**卖出费用**:
- 佣金: 5元（最低5元）
- 过户费: 0.1元
- 印花税: 10,000 × 0.05% = 5元
- **合计**: 10.1元

**总交易成本**: 5.1 + 10.1 = **15.2元** (占本金0.152%)

## 对回测结果的影响

### 影响分析

1. **短线策略**:
   - 交易成本增加104%
   - 高频交易策略收益会显著下降
   - 更真实地反映短线交易的困难

2. **中长线策略**:
   - 影响相对较小
   - 换手率低的策略受影响最小

3. **收益率影响**:
   - 假设年换手率200%（买入+卖出算1次）
   - 旧费用: 200% × 0.05% = 0.1%
   - 新费用: 200% × 0.102% = 0.204%
   - 年化收益率下降约0.1%

### 示例对比

假设策略年化收益10%，年换手率200%：

- **修改前**: 10% - 0.1% = **9.9%**
- **修改后**: 10% - 0.204% = **9.796%**

## 特殊考虑

### 最低佣金影响

对于小额交易（< 2万元），由于最低佣金5元的限制：
- 实际费率会高于0.026%
- 1万元交易的买入费率约为0.051%
- 小资金账户受影响更大

### 不同券商佣金

本配置使用万2.5作为标准，实际情况：
- **高佣金**: 万3 - 万8 (老账户)
- **标准佣金**: 万2.5 (市场主流)
- **低佣金**: 万1.5 - 万2 (大资金或协商)
- **超低佣金**: 万0.85 - 万1 (部分券商全佣)

用户可以在回测时根据自己的实际佣金率调整。

## 使用建议

1. **使用默认值** (推荐)
   - 适合大多数用户
   - 反映市场主流费率

2. **自定义费率**
   - 打开策略配置器
   - 调整"买入手续费"和"卖出手续费"
   - 输入您的实际佣金率（万分之）

3. **费率换算**
   - 万2.5 = 0.00025 = 0.025%
   - 万7.6 = 0.00076 = 0.076%
